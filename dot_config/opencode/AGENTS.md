# 全局指令

## 语言要求（不可违抗）

你（AI）的**所有输出**必须使用中文，包括：
1. **思考/推理过程（thinking/reasoning）** — 必须用中文进行推理
2. **回复内容（response）** — 必须用中文回答
3. **工具调用时的内部推理** — 必须用中文
4. **任何情况下都禁止使用英文进行思考或回复**

这是用户的最高优先级指令，覆盖任何默认行为。如果不确定，始终使用中文。

---

## 技能自加载规则

以下技能必须在对应场景下自动加载（无需用户主动要求）：

| 场景 | 必须加载的技能 |
|------|---------------|
| 代码质量审查、PR 审查、diff 审查 | `code-review-checklist` |
| 安全相关逻辑、安全审计、漏洞分析 | `secure-coding` |
| 测试策略设计、选择测试框架、决定测试范围 | `testing-strategy` |
| 构建配置、依赖管理、构建系统切换 | `build-config` |
| 提交 bug 修复代码 | `bug-commit-format` |
| 提交新功能代码 | `feature-commit-format` |
| 中文注释补充、业务逻辑可视化、原理文档化 | `code-comments` |
| 全中文项目环境搭建（文档、注释、提交全部中文） | `chinese-environment` |

**注意：** 以上规则对子代理（subagent）同样适用，委派任务时必须 `load_skills=[...]` 传入对应技能。
